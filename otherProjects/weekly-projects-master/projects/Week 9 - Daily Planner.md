# Daily Planner

An app where the user can create tasks and organize them by priority or status.

## User Stories

-   [ ] User can create a task.
-   [ ] Task has status and priority.
-   [ ] User can sort tasks by status and priority.

## Bonus features

-   [ ] Drag & Drop.
-   [ ] Search function.
-   [ ] Due Dates.
-   [ ] Attachments.
